## 🚀 快速启动指南

### 1. 安装依赖
```bash
pip install django pillow
```

### 2. 应用数据库迁移
```bash
python manage.py migrate
```

### 3. 启动开发服务器
```bash
python manage.py runserver
```

### 4. 访问系统
- 首页：http://127.0.0.1:8000/
- 管理后台：http://127.0.0.1:8000/admin/
